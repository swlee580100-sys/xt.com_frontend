# 2025-11-10 新增功能說明

## CMS 模組
- 新增「分享平台文案設置」分頁，可自訂分享文案並支援恢復預設值。
- 新增「入金地址設置」，提供後台調整顯示入金地址的介面。

## 客服管理
- 側邊欄新增「客服管理」入口，提供對話列表與訊息回覆功能。
- 支援單筆訊息附檔（圖片 / 影片），並顯示交易摘要及倒數資訊。

## 開盤設置
- 大盤可設定預設輸贏策略（全贏 / 全輸 / 隨機 / 設置買漲或買跌 / 個別用戶）。
- 小盤詳情改為左右雙欄：左側顯示小盤列表，右側提供交易紀錄、倒數與統計。
- 小盤支援個別輸贏策略切換，含「個別用戶設置」模式；調整單一用戶時不會影響其他用戶。
- 新增交易方向 Switch 控制，並於全贏 / 全輸策略下自動同步所有用戶。
- 小盤編輯視窗可設定輸贏策略與單一勝出方向。

## 使用介面改進
- 側邊欄新增收合功能，縮成圖示時仍可操作。
- 新增使用者詳細頁面，提供交易清單、編輯功能及統計。
- 開盤方案對話框高度縮小並支援垂直滾動。
